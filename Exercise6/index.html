Editor Features

A wealth of functionality that can slot in and out of your workflow.

Text, code, and binary edit

Text, code, binary, log files, and more. Use a multitude of features to find, sort, arrange, and manipulate text data in any way.

Find and replace

Search and replace text in one or more text files using customized filters, targeting, and pattern matching techniques. Supports searching specific file types, project files, directories of files.

File compare

Compare files or folders from within UltraEdit easily with native integration of UltraCompareâ€”our powerful compare, merge, and sync tool.

XML manager/View

Navigate through huge set of data or many nested nodes. Easily parse an XML tree, search, move nodes up and down, copy or cut nodes, and even duplicate with XML manager.

Web development

Build websites with UE. Supports many web languages, an HTML toolbar, CSS style builder, and a tag list for easy access to frequently used code.

Custom language support

Work with any language or markup language with vast syntax highlighting support. Create and share your custom wordfiles or get one from the community library.

Native FTP/SFT/FTPS

Access, modify, and save files via FTP. UltraEdit and UEStudio support standard FTP and SFTP protocols as well as FTPS (FTP over SSL).

Custom Themes

Control of the appearance of the application: from syntax highlighting, menus to dockable panes, from toolbars and editor colors to status bar styling.

Scripting and Macros

Simplify and automate repetitive tasks with scripts or macros. Record keystrokes or program instructions for faster and easier text processing.
